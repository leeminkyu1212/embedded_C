비트 clear
• 특정 비트를 0 으로 만드는 것을 clear 한다고 표현한다.
비트 set
• 특정 비트를 1로 만드는 것을 set 한다라고 한다.



unsigned char a= 0b11000000;
//3번 비트 set
unsigned char a3set= a | (0x01 <<3);
//5번 비트 set
unsigned char a5set= a | (0x01 <<5);
//0번 비트 set
unsigned char a0set= a | (0x01 <<0);
//2번 bit, clear 하기
unsigned char a2clr = a & ~(1 << 2);
